# Menú Radial #
Este menú simula un disco de vinilo que gira para mostrar sus opciones.
## Estados: ##
1. Cerrado / Colapsado: Las opciones no están visibles y el disco es pequeño para no ocupar mucho espacio en pantalla.
2. Hover: Se escala su tamaño un poco y se le sobrepone un circulo con color.
3. Abierto / Visible: Las opciones ya son visibles, el tamaño del disco pasa a ser casi equivalente al alto de la pantalla. 
   Se puede girar con la rueda del ratón, con las flechas del teclado o con gestos tactiles para mostrar los enlaces que contiene.
